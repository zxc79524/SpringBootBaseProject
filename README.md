## 執行方式

編譯打包完成後執行,jar檔名稱依照打包結果自行替換

```shell
java -jar SpringBootBaseProject-0.0.1.jar
```

打開瀏覽器輸入

```shell
http://127.0.0.1:8080/
```

Swagger 位置

```shell
http://localhost:8080/swagger-ui/index.html
```
